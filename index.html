Projetos

Meu Player de Música Pessoal!

Criei um player de áudio personalizado usando HTML, CSS e JavaScript! A ideia foi montar uma interface bem limpa e direto ao ponto para ouvir minhas faixas favoritas. Foi um projeto super divertido de fazer e excelente para colocar em prática a manipulação do DOM e deixar o visual responsivo.

Meu Projeto Pokémon

Montei uma página em HTML para listar alguns Pokémon de um jeito bem maneiro! O foco foi mostrar os bichos na tela junto com os status de cada um: Força, Velocidade, Peso e Habilidade. Ficou um visual bem direto ao ponto e foi massa demais de estruturar!

Meu Próprio Mini RPG em HTML!

Criei um mini jogo de RPG direto no navegador! Montei um painel de batalha onde dá para controlar o status do personagem ("Salles 🌵"): ela ganha nível, toma dano com barra de vida interativa e usa poção para se curar. O código ficou bem dinâmico e foi massa demais ver a lógica de batalha funcionando na tela!
